About The Project




This project is an AI-Powered Mobile Phone Recommender that helps users find the best mobiles based on natural language queries.

Features include:
1. Natural Language Search – e.g., "Best 8GB RAM phone under 25000".
2. Semantic Search – powered by NLP embeddings and vector similarity.
3. Interactive Streamlit UI – fast, clean, and scalable.



Project Details!

Mobile Recommender System


This intelligent system suggests the best phones based on your requirements. Whether you’re looking for a device with great camera quality, long battery life, or powerful performance, the system analyzes your preferences and provides the most suitable options.

Technologies used: Python, Streamlit, ChromaDB, Sentence Transformers, Pandas.

Skills demonstrated: NLP, Vector Search, Data Processing, Deployment

In addition to recommending phones, the system leverages embeddings and semantic similarity search to understand natural language queries. For example, instead of manually filtering specifications, you can simply type “best phone under 20k with good camera” and instantly get suggestions. The project also showcases practical deployment of AI-driven applications using Streamlit, making it interactive and user-friendly.

Recommender System


Just like how Netflix recommends movies, this recommender system focuses on mobile phones instead of movies. It helps users find devices similar to the ones they already like.

For example, if a user selects a particular phone model, the system finds and recommends similar phones based on specifications, features, and performance benchmarks. This approach is particularly useful for users who already have a preferred brand or model in mind but want to explore alternatives.

Technologies used: Python, Machine Learning Algorithms, Vector Similarity Search.

Skills demonstrated: Machine Learning Algorithms, Python, Deployment

This project not only demonstrates machine learning techniques for recommendation but also bridges the gap between natural language understanding and structured data filtering. It’s a practical demonstration of how recommender systems can be applied outside of entertainment (like movies or music) into consumer electronics to improve decision-making.
